# VIKA - Vitamin Companion App

VIKA adalah aplikasi pendamping vitamin cerdas yang dirancang untuk membantu pengguna mengelola konsumsi suplemen harian dengan lebih efektif. Banyak orang mengonsumsi berbagai jenis vitamin tanpa mengetahui aturan waktu yang tepat, serta interaksi zat antar vitamin dengan makanan atau minuman. VIKA hadir sebagai solusi untuk memberikan panduan jadwal yang terstruktur, rekomendasi pendamping yang aman, serta pemantauan progres kesehatan yang personal.

## 🚀 Fitur Utama
- **Vitamin Scheduler:** Mengatur jadwal konsumsi vitamin Anda agar tidak terlewatkan dan memastikan durasi antar konsumsi optimal.
- **Smart Compatibility Checker:** Memberikan informasi mengenai zat apa yang boleh atau tidak boleh dikonsumsi bersamaan dengan vitamin tertentu (interaksi obat & makanan).
- **Food & Drink Pairing:** Memberikan rekomendasi makanan atau minuman pendamping yang tepat untuk meningkatkan penyerapan vitamin.
- **Insights & Progress Tracking:** Visualisasi data untuk memantau sejauh mana target konsumsi vitamin Anda telah terpenuhi dan memonitor efektivitas suplemen bagi kesehatan tubuh secara berkala.

## 📱 Cara Menjalankan
1. Clone repositori ini dengan perintah: git clone https://github.com/siwiforestry/vika.git
2. Pastikan Flutter SDK sudah terinstal di komputer Anda.
3. Masuk ke direktori proyek: cd vika
4. Unduh semua dependensi yang diperlukan: flutter pub get
5. Jalankan aplikasi di emulator atau perangkat fisik Anda: flutter run
